Ventana Principal



Barra de Menú        

La Barra de menú le permite acceder a todas las funcionalidades del sistema. Las funciones estan agrupadas por Modulos.


Accesos Rápidos

La Barra de Accesos rapidos le permite un acceso mucho mas flexible que la barra de Menú a las distintas funcionalidades del sistema. Para hacer que la barra de Accesos Rápidos se oculte automaticamente, seleccione el icono "Ocultar automaticamente" .

Si la barra esta oculta deslice el mouse sobre la barra para que esta aparezca. Si desea que la barra permanezca visible haga un click sobre el icono .


Barra de Tareas

La Barra de Tareas contiene enlaces a todas la ventas abiertas. Esto permite hacer visibles una venta que ha sido tapadas por otras ventanas .

Para traer al frente una ventana hago un click sobre la ventana deseada.


Area de trabajo

El area de trabajo es el lugar donde son abiertas las distintas ventanas cuando se ejecuta una funcionalidad del sistema.

En el area de trabajo solo es posible hacer visible una ventana al mismo tiempo. Para cambiar de ventana utilice la barra de tareas.


Mensajes Nuevos

La ventana Principal cuenta con un acceso a la buzón de mails. En la misma se puede observar si tiene mensajes nuevos en su cuenta de correo electronico.
