Gestión del Pedido de Venta

Gestion del Pedido de Ventas comprende:


Creación de un Pedido de Ventas.

Esta funcionalidad permite ingresar una nueva solicitad de un cliente. Este es el punto de partida a la planificación de la producción.


Modificación de un Pedido de Ventas

La Modificación de un pedido se realiza cuando se desea modificar un pedido ya creado. Si el estado del pedido es "FINALIZADO" o "EN PRODUCCION" la modificación del pedido no podrá realizarse.


Anulación de un Pedido de Ventas

La anulación del pedido se realizará cuando se desee va de baja una solicitud del cliente totalmente. Si el estado del pedido es "FINALIZADO" la anulación del pedido no podrá realizarse.
