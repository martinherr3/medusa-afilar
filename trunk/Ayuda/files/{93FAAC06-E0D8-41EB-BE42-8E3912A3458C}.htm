Para ingresar al sistema MEDUSA es mandatorio logearse al mismo ingresando un nombre de usuario y contraseña válidos.


                       .


Pasos a seguir para entrar al sistema:

  1. Ingrese el Nombre el nombre de usuario y contraseña y presione Aceptar.
  2. A continuacion el sistema validará que el usuario y la contraseña ingresadas sean validos. Si los datos ingresados son correctos ingresará al sistema y podrá utilizar todas las funciones disponibles segun el perfil del usuario logeado, caso contrario deberá probar con un usuario y contraseñas válidos para logearse exitosamente al sistema.
